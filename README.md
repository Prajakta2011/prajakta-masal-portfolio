# Prajakta Masal - Professional Portfolio

This repository contains the source code for my professional portfolio website, hosted at [https://Prajakta2011.github.io/prajakta-masal-portfolio/](https://Prajakta2011.github.io/prajakta-masal-portfolio/).

## 🚀 Features
- **Responsive Design**: Built with Tailwind CSS for a seamless experience across all devices.
- **Interactive Elements**: Includes a custom particle background using HTML5 Canvas.
- **Project Showcase**: Highlights my key projects with links to their respective repositories.
- **Professional Details**: Includes my work experience at Saama, education, and certifications.

## 🛠️ Tech Stack
- **HTML5**
- **Tailwind CSS**
- **JavaScript** (Vanilla)
- **Google Fonts** (Inter)

## 📁 Repository Structure
- `index.html`: The main entry point for the portfolio.
- `.git/`: Git version control files.

I am a results-driven Data Engineering professional and Computer Engineering graduate with a specialized Post Graduate Diploma in Big Data Analytics (CDAC-DBDA). Currently a Data Engineer at Saama, I specialize in architecting cloud-native data solutions using the Reltio Multidomain MDM platform.
